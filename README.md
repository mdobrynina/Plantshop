# Moh

Интернет-магазин комнатных растений.

## Стек

- **Frontend:** React 18 + JavaScript + классический CSS, сборка через Vite
- **Backend:** Java Spring Boot (будет добавлен позже)
- **БД:** PostgreSQL (будет добавлена позже)

## Структура

```
Moh/
├── frontend/   ← React + Vite
├── backend/    ← Spring Boot (TODO)
└── README.md
```

## Запуск фронтенда

Один раз — установить зависимости:
```powershell
cd frontend
npm install
```

Запустить дев-сервер:
```powershell
npm run dev
```

Откроется http://localhost:5173
